## Week2-3 Github, fork, pull request
* 포크를 통해 연동하기
  > 내 리파지터리가 다른 사람의 리파지터리로 부터 데이터가 넘어왔는가?
* 풀 리퀘스트를 통해 기여하라
  * 과제의 내용
    1. 다른 사람의 리파지터리에 내 파일을 _리퀘스트_ 한다.
    2. 이후 **_merge_**된 내용을 카피한다.
    3. 메모장에 작성 후 파일 이름을 잘 확인한다.
    4. 전남대 이클래스에 업로드 한다.
    >과제의 이름은 무조건 지킬 것!
